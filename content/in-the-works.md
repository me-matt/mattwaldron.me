<p>PMF: Probability mass function. Evaluated at a value and corresponds to the
probability that a random variable takes that value. To be a valid PMF, a
function p, must satisfy:</p>
<ol>
<li>Must always be > 0</li>
<li>The sum of possible values that random variable can take must add to 1.</li>
</ol>
</li>
<li>
<p>PDF: Probability density function. A function associated with a continuous
random variable.</p>
<ol>
<li>Must be >= 0 everywhere.</li>
<li>Total area uner must be 1.</li>
</ol>
</li>
